## Key Decisions / Constraints

- **Default output under `/tmp/`** — avoids repo/vault pollution and write conflicts with other skills.
- **Optional `goal` argument** — omitted goal triggers default behavior; when present, it acts as a high-alignment anchor.
- **Two plans (agent decides):** Plan A (direct distill) for clean transcripts; Plan B (extract → distill) for tool-heavy sessions.
- **Propose-only by default** — no file writes unless `--apply` is explicitly passed.
- **Hard safety rules:** never delete/rewrite source session history; do not assume dialogue is about code; do not force a rigid note structure.
- **Filtering:** user text and assistant text blocks are kept; toolCall/toolResult noise is stripped in the exported view.
