# intent-writer help

## What it does

Distills the current session into a minimal `INTENT.md` (Goal + Target state invariants).

## Usage

- `/intent-writer` — prints an `INTENT.md` proposal

## Output format

```md
# <title>

Model: gpt-5.2
Updated: <YYYY-MM-DD>

## Goal
...

## Target state (invariants)
- ...
```
